# Бой на основе двух "стеков" юнитов
### Применены паттерны
Порождающие паттерны
- Фабричный метод (Factory Method)
- Абстрактная фабрика (Abstract Factory)
- Прототип (Prototype)

Паттерны поведения
- Стратегия (Strategy)
- Наблюдатель (Observer)
- Команда (Command)

Структурные паттерны
- Декоратор (Decorator)
- Адаптер (Adapter)
